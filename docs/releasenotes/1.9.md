# Release Notes for Version 1.9

Released 2026-01-27.

- It is now possible to install and update Software Accounting from PyPI.
- New sampler module SlurmCGroup2 with support for CGroups version 2.
- New listener module Prometheus.
- Add most recent sample functionality to samplers.
- Add nvidia_smi averaging function.
- Include JobName in SlurmInfo sampler.
- Upgraded bundled copy of sgas-sa-registrant to latest version.
- Bug fixes and various improvements to the build system.

Upgrading from version 1.8 does not require any changes to configurations files.
