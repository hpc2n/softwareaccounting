# Release Notes for Version 1.10

Released 2026-03-03.

- The bundled copy of sgas-sa-registrant has been updated. By default the container tag will now be SoftwareAccountingRecords. This was expected by SAMS but had to be set explicitly in the configuration file.

Upgrading from version 1.9 require changes to configurations files only if the default records_tag configuration of sgas-sa-registrant has been used.
